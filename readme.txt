Instructions:
1. Clone the repository locally in whatever repo you wish.
2. Create a virtual environment and activate it.
3. run the following command to install the necessary libraries: 'pip install numpy scipy'
4. run the command by calling 'python text_transmission_decode.py' from the directory (You need to be in the same directory for this command to work)

In order to to use out program, go to line 138 and 139. 
At line 138, input the path of the input.txt file you want to decode
At line 139, input the path of the preamble.txt file you want to decode 

After, on vscode, hit run. ( if you are using CMDline, cd to the program location, then type "python text_transmission_decode.py" to run" )
